# MVP Plan

## Goal

Ship a usable desktop RPA alpha that supports:

- Building flows in a low-code studio
- Running flows locally
- Triggering flows manually, by schedule, or from file changes
- Handling web automation, file processing, and Excel-style data tasks

## MVP modules

### 1. App Center

- Create, rename, duplicate, archive, and open automation apps
- Show last modified time, status, and latest run summary

### 2. Flow Editor

- Left instruction tree with built-in categories
- Center structured flow canvas
- Right panel for variables, resources, and flow outline
- Bottom panel for element library, image library, run logs, and data tables

### 3. Execution

- Manual run
- Step-by-step debug
- Breakpoints
- Variable inspection
- Failure screenshots and error details

### 4. Triggers

- Manual
- Schedule
- File change

## First instruction packs

- Flow control
- Variables and data
- Web automation
- Desktop basics
- File operations
- Excel/CSV operations
- HTTP/network basics

## Success criteria

- A user can create a file-triggered flow that reads a local file, opens a webpage, fills a form, and writes results back to disk.
- A user can debug the flow and understand which step failed.
